## 3. 画面詳細

### SC-001: ログイン画面

#### 3.1.1 画面概要

- **目的**: ユーザー認証を行い、システムへのアクセスを許可する
- **表示タイミング**: システムアクセス時、またはセッションタイムアウト時

#### 3.1.2 画面レイアウト

```
┌─────────────────────────────────────┐
│     営業日報システム                   │
│                                     │
│     ┌─────────────────────────┐    │
│     │ メールアドレス              │    │
│     │ [________________]        │    │
│     │                           │    │
│     │ パスワード                 │    │
│     │ [________________]        │    │
│     │                           │    │
│     │ [  ログイン  ]             │    │
│     │                           │    │
│     │ □ ログイン状態を保持する    │    │
│     └─────────────────────────┘    │
│                                     │
│     パスワードをお忘れの方はこちら     │
└─────────────────────────────────────┘
```

#### 3.1.3 入力項目

| 項目名                 | 形式             | 必須 | 最大長 | バリデーション     |
| ---------------------- | ---------------- | ---- | ------ | ------------------ |
| メールアドレス         | テキスト         | ◯    | 255    | メールアドレス形式 |
| パスワード             | パスワード       | ◯    | 128    | -                  |
| ログイン状態を保持する | チェックボックス | -    | -      | -                  |

#### 3.1.4 ボタン・アクション

| ボタン名                       | アクション                 | 遷移先               |
| ------------------------------ | -------------------------- | -------------------- |
| ログイン                       | 認証処理を実行             | SC-002: 日報一覧画面 |
| パスワードをお忘れの方はこちら | パスワード再設定メール送信 | -                    |

#### 3.1.5 バリデーション・エラーメッセージ

| 条件                 | エラーメッセージ                                     |
| -------------------- | ---------------------------------------------------- |
| メールアドレス未入力 | 「メールアドレスを入力してください」                 |
| パスワード未入力     | 「パスワードを入力してください」                     |
| 認証失敗             | 「メールアドレスまたはパスワードが正しくありません」 |

### SC-002: 日報一覧画面

#### 3.2.1 画面概要

- **目的**: 日報の一覧を表示し、検索・閲覧・新規作成を行う
- **表示タイミング**: ログイン後、または日報詳細画面から戻る時

#### 3.2.2 画面レイアウト

```
┌─────────────────────────────────────────────────────────────┐
│ [ロゴ] 営業日報システム                   [ユーザー名▼] [ログアウト] │
├─────────────────────────────────────────────────────────────┤
│ ■ 日報一覧  □ 顧客マスタ                                      │
├─────────────────────────────────────────────────────────────┤
│ 検索条件                                                      │
│ ┌──────────────────────────────────────────────────────┐   │
│ │ 表示対象: (○) 自分の日報  ( ) 部下の日報  ( ) 全て       │   │
│ │ 報告日: [2026/01/01] ～ [2026/02/01]                  │   │
│ │ 営業担当者: [___選択___▼]  顧客: [___選択___▼]         │   │
│ │ [  検索  ] [  クリア  ]                               │   │
│ └──────────────────────────────────────────────────────┘   │
│                                                               │
│ [ + 新規日報作成 ]                           全50件 (1/5ページ) │
│                                                               │
│ ┌──────────────────────────────────────────────────────┐   │
│ │報告日    │営業担当者│訪問件数│Problem│Plan│最終更新    │    │   │
│ ├──────────────────────────────────────────────────────┤   │
│ │2026/02/01│山田 太郎│   3件 │  2件 │ 2件│2026/02/01 18:30│ │   │
│ │2026/01/31│山田 太郎│   2件 │  1件 │ 1件│2026/01/31 19:00│ │   │
│ │2026/01/30│山田 太郎│   4件 │  3件 │ 2件│2026/01/30 18:45│💬│   │
│ │2026/01/29│山田 太郎│   1件 │  1件 │ 1件│2026/01/29 19:15│ │   │
│ │2026/01/28│山田 太郎│   3件 │  2件 │ 3件│2026/01/28 18:20│💬│   │
│ │    ...                                                │   │
│ └──────────────────────────────────────────────────────┘   │
│ [< 前へ] [1] [2] [3] [4] [5] [次へ >]                         │
└─────────────────────────────────────────────────────────────┘

凡例: 💬 = 上長コメントあり
```

#### 3.2.3 表示項目

| 項目名           | 説明                         | ソート |
| ---------------- | ---------------------------- | ------ |
| 報告日           | 日報の対象日                 | ◯      |
| 営業担当者       | 日報作成者の氏名             | ◯      |
| 訪問件数         | その日の訪問記録の件数       | ◯      |
| Problem          | Problemの件数                | ◯      |
| Plan             | Planの件数                   | ◯      |
| 最終更新         | 日報の最終更新日時           | ◯      |
| コメントアイコン | 上長コメントがある場合に表示 | -      |

#### 3.2.4 検索条件

| 項目名       | 形式         | 説明                       |
| ------------ | ------------ | -------------------------- |
| 表示対象     | ラジオボタン | 自分の日報/部下の日報/全て |
| 報告日(From) | 日付         | 検索開始日                 |
| 報告日(To)   | 日付         | 検索終了日                 |
| 営業担当者   | プルダウン   | 営業マスタから選択         |
| 顧客         | プルダウン   | 顧客マスタから選択         |

#### 3.2.5 ボタン・アクション

| ボタン名     | アクション               | 遷移先                     |
| ------------ | ------------------------ | -------------------------- |
| 新規日報作成 | 新規日報作成画面を開く   | SC-003: 日報詳細・編集画面 |
| 検索         | 検索条件で日報を絞り込み | -                          |
| クリア       | 検索条件をクリア         | -                          |
| 行クリック   | 選択した日報の詳細を表示 | SC-003: 日報詳細・編集画面 |

#### 3.2.6 権限による表示制御

| ユーザー種別   | 表示可能な日報          |
| -------------- | ----------------------- |
| 一般営業担当者 | 自分の日報のみ          |
| 上長(課長以上) | 自分の日報 + 部下の日報 |
| 管理者         | 全ての日報              |

#### 3.3.3 表示項目

**基本情報**
| 項目名 | 形式 | 編集可否 | 説明 |
|-------|-----|---------|------|
| 報告日 | 日付選択 | ◯(作成者のみ) | 日報の対象日 |
| 営業担当者 | テキスト(表示のみ) | × | 作成者の氏名 |

**訪問記録セクション**

- 訪問記録のリスト表示
- 顧客名、訪問時間、訪問内容を表示
- 編集・削除ボタン

**Problem/Planセクション**

- 内容の表示
- 上長コメントの表示・追加(上長のみ)
- 編集・削除ボタン(作成者のみ)

#### 3.3.4 ボタン・アクション

| ボタン名        | 表示条件   | アクション                         | 遷移先 |
| --------------- | ---------- | ---------------------------------- | ------ |
| 一覧に戻る      | 常時       | 一覧画面に戻る                     | SC-002 |
| 訪問記録 - 追加 | 作成者のみ | 訪問記録登録ダイアログを開く       | SC-004 |
| 訪問記録 - 編集 | 作成者のみ | 訪問記録編集ダイアログを開く       | SC-004 |
| 訪問記録 - 削除 | 作成者のみ | 訪問記録を削除(確認ダイアログ表示) | -      |
| Problem - 追加  | 作成者のみ | Problem登録ダイアログを開く        | SC-005 |
| Problem - 編集  | 作成者のみ | Problem編集ダイアログを開く        | SC-005 |
| Problem - 削除  | 作成者のみ | Problemを削除(確認ダイアログ表示)  | -      |
| Plan - 追加     | 作成者のみ | Plan登録ダイアログを開く           | SC-006 |
| Plan - 編集     | 作成者のみ | Plan編集ダイアログを開く           | SC-006 |
| Plan - 削除     | 作成者のみ | Planを削除(確認ダイアログ表示)     | -      |
| コメント表示    | 上長のみ   | コメント入力欄を展開               | -      |
| 保存            | 作成者のみ | 日報を保存                         | -      |
| 削除            | 作成者のみ | 日報を削除(確認ダイアログ表示)     | SC-002 |

#### 3.3.5 権限による表示制御

| 要素             | 作成者 | 上長           | その他 |
| ---------------- | ------ | -------------- | ------ |
| 編集・削除ボタン | 表示   | 非表示         | 非表示 |
| 追加ボタン       | 表示   | 非表示         | 非表示 |
| コメント入力欄   | 非表示 | 表示           | 非表示 |
| 閲覧             | 可     | 可(部下の日報) | 不可   |

### SC-004: 訪問記録登録・編集ダイアログ

#### 3.4.1 画面概要

- **目的**: 訪問記録の登録・編集を行う
- **表示タイミング**: 日報詳細画面から「訪問記録 - 追加」または「編集」をクリック時

#### 3.4.2 画面レイアウト

```
┌───────────────────────────────────────┐
│ 訪問記録の登録・編集          [×] 閉じる │
├───────────────────────────────────────┤
│                                       │
│ 顧客 *                                │
│ [___選択してください___▼]              │
│ または [新規顧客を登録]                │
│                                       │
│ 訪問開始時刻                           │
│ [10]:[00]                             │
│                                       │
│ 訪問終了時刻                           │
│ [11]:[30]                             │
│                                       │
│ 訪問内容 *                             │
│ ┌─────────────────────────────┐     │
│ │                                 │     │
│ │                                 │     │
│ │                                 │     │
│ │                                 │     │
│ └─────────────────────────────┘     │
│ 0/1000文字                             │
│                                       │
│            [  キャンセル  ] [  保存  ] │
└───────────────────────────────────────┘

* は必須項目
```

#### 3.4.3 入力項目

| 項目名       | 形式           | 必須 | 最大長 | バリデーション            |
| ------------ | -------------- | ---- | ------ | ------------------------- |
| 顧客         | プルダウン     | ◯    | -      | 顧客マスタから選択        |
| 訪問開始時刻 | 時刻選択       | -    | -      | HH:MM形式                 |
| 訪問終了時刻 | 時刻選択       | -    | -      | HH:MM形式、開始時刻より後 |
| 訪問内容     | テキストエリア | ◯    | 1000   | -                         |

#### 3.4.4 ボタン・アクション

| ボタン名       | アクション                           |
| -------------- | ------------------------------------ |
| 新規顧客を登録 | 顧客マスタ登録画面を別タブで開く     |
| キャンセル     | ダイアログを閉じる(変更は破棄)       |
| 保存           | 訪問記録を保存してダイアログを閉じる |

#### 3.4.5 バリデーション・エラーメッセージ

| 条件                     | エラーメッセージ                                   |
| ------------------------ | -------------------------------------------------- |
| 顧客未選択               | 「顧客を選択してください」                         |
| 訪問内容未入力           | 「訪問内容を入力してください」                     |
| 終了時刻が開始時刻より前 | 「訪問終了時刻は開始時刻より後に設定してください」 |

## 4. 共通仕様

### 4.1 ヘッダー

全画面共通のヘッダーを配置

| 要素           | 説明                                                   |
| -------------- | ------------------------------------------------------ |
| ロゴ           | システムロゴ(クリックで日報一覧へ)                     |
| ナビゲーション | 日報一覧、顧客マスタへのリンク                         |
| ユーザー名     | ログイン中のユーザー名(プルダウンでプロフィール・設定) |
| ログアウト     | ログアウト処理                                         |

### 4.2 ページネーション

一覧画面では10件/ページで表示し、ページネーションを実装

### 4.3 レスポンシブ対応

- PC: 1200px以上
- タブレット: 768px-1199px
- スマートフォン: 767px以下

スマートフォン表示時は、一覧のカラム数を削減し、スクロール可能にする

### 4.4 確認ダイアログ

削除操作時は以下の確認ダイアログを表示

```
┌───────────────────────────┐
│ 確認                       │
├───────────────────────────┤
│ 本当に削除してもよろしいですか?│
│ この操作は取り消せません。   │
│                           │
│   [キャンセル]  [削除する]  │
└───────────────────────────┘
```

### 4.5 エラー表示

- 入力エラー: 該当項目の下に赤文字で表示
- システムエラー: 画面上部に赤背景のアラートボックスで表示
- 成功メッセージ: 画面上部に緑背景のアラートボックスで表示

```
┌─────────────────────────────────────┐
│ ✓ 日報を保存しました                 │
└─────────────────────────────────────┘
```

### 4.6 ローディング表示

データ取得・保存中は画面中央にローディングアニメーションを表示

## 5. 画面設計における考慮事項

### 5.1 ユーザビリティ

- 一覧画面からワンクリックで詳細画面に遷移
- ダイアログでの入力は最小限の項目に絞る
- 必須項目は明確に表示(\*マーク)

### 5.2 アクセシビリティ

- キーボード操作のサポート(Tab、Enter、Escキー)
- コントラスト比の確保(WCAG 2.1 AA準拠)
- スクリーンリーダー対応

### 5.3 パフォーマンス

- 一覧画面はページネーションで分割
- 画像の遅延読み込み
- 検索条件の保持(セッションストレージ)

### 5.4 セキュリティ

- XSS対策(入力値のサニタイズ)
- CSRF対策(トークン)
- 権限チェック(画面表示前にサーバーサイドで実施)
